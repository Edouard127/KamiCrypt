
# KamiCrypt

A very basic image encryption and decryption program made in javascript


Usage: node encrypt.js .\\<source_image>
It can be jpg, jpeg, png or bmp
I only tested those

Usage: node decrypt.js .\\<source_image>_e.jpg .\\<source_image>_key.txt
